# Daily Journal Writer

## Workflow
1. Gather user facts and same-day evidence
2. Keep only top-impact progress and failures
3. Apply `obsidian-markdown`
4. Apply `obsidian-engram-frontmatter`
5. Store reusable lessons in Engram when appropriate

## Guardrails
- Prefer compact outputs
- Avoid duplicate same-day sections
- Do not create extra files unless necessary
