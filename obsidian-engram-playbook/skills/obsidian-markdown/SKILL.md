# Obsidian Markdown

## Activation
- Default-on for Obsidian `.md` work
- Pairs with `obsidian-engram-frontmatter` when metadata matters

## Rules
- Use valid frontmatter
- Use wikilinks for internal notes
- Use standard links for external URLs
- Use callouts and embeds only when they improve readability
- Keep structure stable and low-noise
